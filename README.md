# oc_c_brushup

Following data structures are implemented:
  - Singly linked list
  - Doubly Linked list
  - Doubly circular linked list
  - Stack using array.
  - Queue using array.
  - Hashtable.

To compile :
  gcc <program_name>
  This will create object file named as a.out

To execute :
  Execute this object file as
  ./a.out
